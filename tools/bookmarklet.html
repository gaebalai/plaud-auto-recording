🎣 PLAUD 토큰 자동 추출 — Bookmarklet

PLAUD Web에서 클릭 한 번으로 진짜 동작 토큰을 클립보드에 복사하는 북마클릿. 토큰 만료 시 갱신용으로 가장 빠릅니다.

1. 설치 (한 번만)

아래 파란 버튼을 드래그해서 브라우저의 북마크바에 끌어다 놓으세요:

📋 PLAUD 토큰 추출

북마크바가 안 보이면? Chrome에서 ⌘ ⇧ B 또는 보기 메뉴 → 북마크바 표시.

2. 사용법

1) 본인 Chrome에서 https://web.plaud.ai 접속, 로그인 상태 확인
2) 메인 화면(녹음 목록) 도달한 상태에서 북마크바의 📋 PLAUD 토큰 추출 클릭
3) "🎣 활성화" 알림 → 확인 누르고 페이지를 한 번 클릭하거나 ⌘ R 새로고침
4) "✅ PLAUD 토큰 클립보드 복사됨" 알림 → 클립보드에 진짜 동작 토큰
5) 터미널에서 npm run refresh-token 실행 후 Enter (클립보드 자동 사용)

3. 안전한가요?

북마클릿은 현재 페이지(plaud.ai)에서만 동작하는 JavaScript입니다. 다른 사이트에서 클릭해도 PLAUD API 헤더가 없으니 아무 일도 안 일어납니다.

하는 일은 단 두 가지:

  • window.fetch와 XMLHttpRequest를 wrap해 첫 Authorization: Bearer ... 헤더를 가로챔
  • 잡힌 토큰을 navigator.clipboard로 복사하고 자기 자신을 해제

토큰을 외부 서버로 전송하지 않습니다. 모든 처리는 브라우저 내부.

4. 문제 해결

알림이 안 떠요
북마클릿 클릭 후 페이지를 클릭/새로고침해야 첫 API 호출이 발생합니다. 30초 후에도 안 뜨면 페이지 로그아웃 후 재로그인.
"클립보드 실패"
Chrome이 클립보드 권한을 거부한 경우. 알림 박스에 토큰이 직접 표시되니 그것을 복사하세요.
북마크가 안 끌립니다
Safari에서는 일부 환경에서 드래그 안 됨. Chrome 사용 권장. 또는 우클릭 → "이 링크 북마크에 추가".

5. 대안 — 콘솔 한 줄

북마크 등록이 부담스러우면 콘솔용 스크립트를 매번 붙여넣어도 됩니다. 동일한 결과.
